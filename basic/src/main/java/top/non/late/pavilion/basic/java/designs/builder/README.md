### Builder Pattern
=======================================

## Description
Builder Pattern 一般包含
- director: 指挥者
- builder: 建造者,interface or abstract class(产品规范,需求)
- concreteBuilder: 实际建造者(实际搬砖的)
- product: 产品 (产品)
- customer: 客户
